# Examples

Runnable integration examples for spackle.

## CLI

- [`cli/`](cli/) — placeholder; planned runnable invocation examples.

## WASM (`@a2-ai/spackle-wasm`)

- [`wasm/bun-script/`](wasm/bun-script/) — **runnable**. Minimal Bun script that reads a template, fills it, writes output.
- [`wasm/node-script/`](wasm/node-script/) — planned. Plain Node.js variant of bun-script.
- [`wasm/tanstack-start-server/`](wasm/tanstack-start-server/) — planned. Server-side project generation inside a TanStack Start route.
- [`wasm/nextjs-route/`](wasm/nextjs-route/) — planned. Same, for Next.js.
- [`wasm/browser-preview/`](wasm/browser-preview/) — planned. Fully client-side preview (wasm runs in the browser).

See [`/docs/wasm/recipes/`](../docs/wasm/recipes/) for sketches matching each planned example.
